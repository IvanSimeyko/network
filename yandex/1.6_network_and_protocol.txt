Утилиты Линукс для настройки сети:
ip link show dev eth0 - выведет мак-адрес на канальном уровне (link)
ip link - выведет мак-адрес всех интерфейсов
ip a - выведет адреса на всех уровнях (мак и айпи)
ip -4 a - выведет только айпи адреса 4-го уровня
ip rout - выведет все маршруты по-умолчанию

netstat - утилита для проверки сетевой конфигурации и активности.
iptables - фильтрация пакетов.

mtr mirror ivansimeyko.com  - посмотреть сколько хабов нужно пройти чтобы достичь указанного адреса

nmap ivansimeyko.com  - покажет открытые порты на сервере ivansimeyko.com . Например сейчас выводит:
PORT   STATE SERVICE
22/tcp open  ssh
80/tcp open  http
81/tcp open  hosts2-ns

tcpdump - аннализирует только заголовки пакетов
